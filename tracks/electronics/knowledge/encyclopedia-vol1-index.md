# Encyclopedia of Electronic Circuits — Vol. 1

**Author:** Rudolf F. Graf
**File:** [Encyclopedia-of-Electronic-Circuits-Vol.1.pdf](Encyclopedia-of-Electronic-Circuits-Vol.1.pdf)

Reference book — circuit cookbook organized by category. Use as a lookup when you need a working schematic for a specific function. Each chapter starts with a list of the circuits in it; each circuit has a `Fig. X-Y` number that maps to a source in the Sources section (printed p. 730).

> Page links open the PDF at the correct PDF page. Printed page 1 lives on PDF page 7, so PDF page = printed page + 6. Front matter uses roman numerals.
